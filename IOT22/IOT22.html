Network Scanning Using Python on a Raspberry Pi




100 Points
Learning Objectives

These instructions will show you how a Raspberry Pi and Python code can scan and monitor a network.
Only scan networks for which you have permission. Do not act maliciously-this assignment is for educational purposes.

More information on the Network Scan library that you will use is available online.

After completing this assignment you will be able to:

  1. Investigate Python libraries
  2. Run Python programs on a Raspberry Pi that perform special purpose network functions such as scanning
  3. Discuss uses of Raspberry Pi as devices to monitor network status

Exercise 1-Python Network Scan Library
  1. These steps are performed on your computer.
  2. Open a browser.
  3. Browse to the following webpage: networkscan which is located in the pypi.org website.
  4. Review the information provided here.
  5. Use VNC to transfer the file ' scannetwork.py ' (available at .txt-rename to .py) from your computer to the Raspberry Pi, and place it in the folder '/home/pi/Desktop/IoT_Files'
    Note- a copy can be found in the OneDrive - MNSCU/ReadOnly/IoT/IoT_PythonFiles folder.
Exercise 2-Scan a Network
  1. These steps are performed on your Raspberry Pi.
  2. Launch a Terminal window
  3. Run the following command to ensure the Python Package installer is present.
    sudo apt-get install python3-pip
  4. Run the following command to ensure that the Networkscan library is installed.
    pip3 install networkscan
  5. Launch Thonny
  6. Load the program /home/pi/Desktop/IoT_Files/scannetwork.py
  7. In the program, on about line 12, adjust the ip address of the network to scan.
    Remember that VNC provides the network address of your Raspberry Pi. Do a /24 scan.

    Rasbberry Pi Network

  8. Run the program scannetwork.py
  9. Examine the output shell to make sure that the network scan data appears, and is reasonable.
  10. Review the program scannetwork.py and think about improvements that could be made.
